# 德新行政管理平台 — Demo

聯新台北運醫會館（運動醫學會館 7F）設施管理示範平台。

- 純前端 vanilla HTML / CSS / JS，零外部依賴，可直接以 `file://` 或任意靜態主機開啟
- 中控戰情總覽 + 16 子系統模組（電力／空調／新風／消防／給排水／資產／物料／門禁／排程／派工／CCTV／安防／人員／回報／供應商／Hermes AI 分流）
- 空間編碼：`7-[區][房]-[子系統]-[點位]`，區域 A–E

## 線上 demo

GitHub Pages：見 repo 的 Pages 設定頁網址。

## 本機開啟

直接用瀏覽器開 `index.html` 即可，無需安裝任何套件或啟動伺服器。
